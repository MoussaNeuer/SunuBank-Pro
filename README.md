# 🇸🇳 SunuBank Pro · Application bancaire mobile

SunuBank est une application de gestion de comptes bancaires entièrement repensée dans un style moderne et professionnel, inspirée des couleurs du drapeau sénégalais (vert, jaune, rouge et blanc). Elle offre une expérience utilisateur fluide, des animations soignées et une panoplie de fonctionnalités dignes d’une vraie banque mobile.

L’application est 100 % responsive et fonctionne sur tous les appareils (mobile, tablette, desktop). Elle a été développée en HTML, CSS et JavaScript (transposition d’un exercice Java) et peut être hébergée facilement sur GitHub Pages.

---

## ✨ Fonctionnalités principales

| Fonctionnalité | Description |
|----------------|-------------|
| **🔐 Authentification complète** | Inscription / Connexion avec validation, code PIN 4 chiffres, option biométrique (simulée). |
| **💳 Multi-comptes** | Possibilité d’avoir plusieurs comptes (courant, épargne, bloqué). Bascule rapide via un carousel. |
| **📊 Tableau de bord** | Affichage du solde total, nombre de transactions, dépenses du mois. Graphique d’évolution du solde. |
| **💰 Dépôt / Retrait** | Opérations sécurisées par code PIN. Ajout d’une description. |
| **🔄 Virements** | Virements entre ses propres comptes ou vers d’autres utilisateurs SunuBank. Motif personnalisable. |
| **💳 Cartes virtuelles** | Génération automatique d’une carte avec numéro, date d’expiration, CVV. Possibilité de bloquer/débloquer et de modifier le plafond. |
| **📋 Historique des transactions** | Liste chronologique de toutes les opérations, avec indication du type (dépôt, retrait, virement) et du montant. |
| **📈 Budget mensuel** | Suivi des dépenses par catégorie (Alimentation, Transport, Loisirs) avec barre de progression et alertes visuelles. |
| **💬 Support client (chat)** | Interface de chat avec réponses automatiques simulées. |
| **🔒 Sécurité** | Modification du code PIN, activation/désactivation de la biométrie. |
| **🔔 Notifications push simulées** | Toast notifications pour chaque action importante (connexion, opération, alerte). |
| **🌙 Mode sombre** | Bascule en un clic, adaptation complète de l’interface. |

---

## 🚀 Fonctionnalités avancées

- **Authentification biométrique simulée** : Empreinte digitale (via un bouton dans la boîte de dialogue PIN).
- **Virements vers d’autres utilisateurs** : Recherche parmi les utilisateurs enregistrés.
- **Cartes virtuelles avec plafond ajustable** : Slider interactif pour modifier le plafond de la carte.
- **Graphique d’évolution** : Affiche l’évolution du solde sur les 7 derniers jours (données simulées).
- **Notifications contextuelles** : S’affichent en haut à droite et disparaissent après 3 secondes.
- **Animations CSS** : fadeInUp, scaleIn, slideInRight, transitions fluides.
- **Mode sombre** : Toutes les couleurs s’adaptent automatiquement.

---

## 🛠 Technologies utilisées

- **HTML5** : Structure sémantique.
- **CSS3** : Variables CSS, Flexbox, Grid, animations, responsive design.
- **JavaScript (ES6)** : Programmation orientée objet, classes, modules.
- **Chart.js** : Génération du graphique d’évolution du solde.
- **Font Awesome 6** : Icônes vectorielles (toutes les icônes de l’application).
- **Google Fonts (Inter)** : Police moderne et lisible.
